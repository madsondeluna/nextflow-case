# First Things First - Guia para Iniciantes

Pasta com roteiros completos para quem nunca executou um pipeline Nextflow antes.

---

## Conteúdo

### 1. README.md - Guia Principal
**Comece por aqui!**

Guia completo passo a passo que cobre:
- Instalação do ambiente (Java, Nextflow, Docker)
- Entendimento do pipeline
- Preparação de dados
- Primeira execução com dados de teste
- Execução com seus próprios dados
- Interpretação de resultados
- Solução de problemas comuns

**Tempo estimado:** 30-60 minutos

## Pré-requisitos

Antes de começar, você precisa:
- Computador com macOS, Linux ou Windows (WSL2)
- Conexão com internet
- Pelo menos 8GB de RAM
- Pelo menos 20GB de espaço em disco
- Permissões de administrador
- 30-60 minutos disponíveis

---

## Suporte

Se encontrar problemas:
1. Consulte a seção "Problemas Comuns" no README.md
2. Verifique o checklist.md para garantir que seguiu todos os passos
3. Consulte a documentação completa em `docs/usage.md`
4. Abra uma issue: https://github.com/madsondeluna/nextflow-case/issues
